1Voice-Command Fixes AI-Assisted

Problem: Customer support team reported issues on HarborOmegadevices where the Voice-Command app was failing to call a person by name.

1.1 Root Cause Analysis

The voice service was not able to detect Contact names properly

1.2 Technical Challenge

Issue originated from the Nuance proprietary library, which was a closed source (.jar) file with no available source code

1.3 Initial Investigation

Decompiled the source code, but encountered fully obfuscated code that was difficult to understand

Decompiled Obfuscated Code

Complex obfuscated code structure making manual analysis extremely difficult

Used AI to convert the obfuscated code and generate overridden methods in a new class, successfully resolving the issue

2Deprecated API Modernization AI-Assisted

Challenge: Platform apps originally developed for XP5 project were ported to HarborOmega and other devices, but contained several deprecated SDK APIs.

2.1 Legacy Code Challenge

Several platform apps had been originally developed for XP5 project and later ported to Harbor, Omega, etc. Over the period, several SDK APIs became deprecated

2.2 AI-Powered Solution

Used CLI AI agent to systematically update deprecated APIs for the Talkback app

2.3 Results & Impact

AI successfully identified several deprecated APIs and outdated practices, providing modernized alternatives

AI Analysis Results

AI identified 8 specific deprecated APIs and provided modern alternatives
